## 2. 实现

词法不变。

文法：需要增加 int 函数名()的解析，后面 compound_stmt 之后不变。注意，因为要实现函数定义，所以处理的字符流中就不止一个函数了，我增加了一个Program结构体，其中放入一个函数列表成员，用Vec实现函数的存储。

代码生成：其实改动不大，就是在codegen上位主控函数中启动一个for循环，对每个函数产生对应的汇编，主要是对标签名字的特异化处理。
